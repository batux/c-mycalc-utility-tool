# c-mycalc-utility-tool
Calculator utility tool in C language!


Compile C Project

```console
$ gcc -Wall -o mycalc mycalc.c 
```

Run C utility program in shell environment (example: bash)

```console
$ ./mycalc
given no option!

$ ./mycalc -d -m 1000 100 20 2
cannot be utilized with multiple options...

$ ./mycalc -d 1000 100 20 2
RESULT: 0.25 

$ ./mycalc -m 1000 100 20 2
RESULT: 4000000.00 

$ ./mycalc -s 1000 100 20 2
RESULT: 878.00 

$ ./mycalc -a 1000 100 20 2
RESULT: 1122.00 
```
